## MixFormer PyTorch 实现

- 从零实现了论文 "MixFormer: Co-Scaling Up Dense and Sequence in Industrial Recommenders" (arXiv:2602.14110) 的完整 PyTorch 版本
- 项目路径: `/Users/yyhaker/project/project/MixFormer`
- 核心模块: config.py, modules.py, layers.py, model.py, data.py
- 脚本: train.py (训练), inference.py (推理)
- 测试: 46 个单元测试全部通过
- 支持 MixFormer 和 UIMixFormer (解耦变体) 两种模型
- 支持 small (N=16, L=4, D=386) 和 medium (N=16, L=4, D=768) 两种预设配置
- MixFormer-small 约 567M 参数

## TorchRec + Alibaba 数据集重构

- 使用 Meta 开源的 TorchRec 重构了 MixFormer 的嵌入层管理
- 集成了 DIN 论文 (1706.06978) 中提到的 Alibaba UserBehavior 数据集 (天池)
  - 数据集: ~1亿条用户行为记录 (user_id, item_id, category_id, behavior_type, timestamp)
  - 天池下载: https://tianchi.aliyun.com/dataset/649
- 新增文件:
  - `scripts/download_alibaba.py`: 数据下载与预处理脚本（支持真实数据和模拟数据）
  - `mixformer/model.py`: 新增 `TorchRecFeatureEncoder`, `AlibabaMixFormer`, `EmbeddingBagFallback`
  - `mixformer/data.py`: 新增 `AlibabaDataset`, `alibaba_collate_fn`, `create_alibaba_dataloader`
  - `mixformer/config.py`: 新增 `alibaba()` 工厂方法
- 重构要点:
  - 用户历史行为序列 → MixFormer CrossAttention 的序列输入 (KV)
  - 目标物品 (item_id + cate_id) → TorchRec EmbeddingBag → MLP → Query 输入
  - TorchRec 不可用时自动 fallback 到 nn.EmbeddingBag
  - Alibaba 配置: N=8, L=3, D=64, 约 6.7M 参数
- 端到端验证通过: 模拟数据生成 → 训练 1 epoch (Loss 0.758→0.700) → 推理 (10000 样本, 2982 samples/s)
- 完全向后兼容: 原有合成数据集训练路径不受影响
